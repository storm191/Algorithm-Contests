Problems

3. Cyclic permutations

Given two strings determine if one is a cyclic permutation of the other. For example: abcde and deabc are cyclic permutations.

Input

The first line is T(<=20) the number of test cases. For each test case, there are two lines, representing the first and the second string respectively. The lengths of the strings are <= 100000.

Ouput

For each test case output 1 if the second string is a cyclic permutation of the first, 0 otherwise.

Sample Input

2 abcde deabc hello olleh

Sample Output

1 0

Time Limit : 1 second

Memory Limit : 64 MB
