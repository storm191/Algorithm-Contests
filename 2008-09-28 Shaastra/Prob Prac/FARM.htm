Problems

2. Farm Harvest

A plantation has several trees, each having a given number of fruits. The trees are in an MxN grid, where one position in the tree has exactly one tree. You are given the arrangement of the trees and the number of fruits each tree has. The farmer wants to clear certain rectangular sections of the land to harvest fruit. He is exploring several options for choosing the section he wants to harvest. Given the section he wants to cut, find the number of fruits he can harvest.

Input

The first line contains T, M, N (T, M, N <= 1000) where T is the number of sections, M and N are the number of rows and columns in the grid. The following M lines have N integers each containing the number of fruits of the corresponding tree in the grid. The next T lines contain 4 integers each R1 (1<=R1<=M), C1(1<=C1<=N), R2(1<=R2<=M), C2(1<=C2<=N) representing the top-left and bottom-right points of the rectangular section.

Ouput

For each section, output the number of fruits in that section. You may assume total number of all fruits in the plantation fits in a 32-bit signed integer.

Sample Input

5 5 5 3 1 2 0 3 0 1 2 4 1 2 2 0 4 3 1 0 1 2 1 1 3 2 4 2 1 3 3 4 1 3 5 3 4 3 5 4 2 3 2 5 2 5 4 5

Sample Output

12 7 9 7 5

Time Limit : 1 second

Memory Limit : 64 MB
