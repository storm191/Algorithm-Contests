Problems

1. Code enumeration

A password is encrypted into N strings. Each string may have Ai different kind of characters and has length exactly Li. Given these values, find the number of possible password encryptions. Report your answer modulo M where M is given.

Input

The first line is T, the number of test cases. Each test case starts with two integers N (1<=N<=100) and M. The next N lines contain two pairs of integers each Ai and Li where Ai <=1000 and Li<=10000.

Ouput

For each test case output the number of password encryptions modulo M.

Sample Input

1 3 5 3 1 2 2 4 3

Sample Output

3

Time Limit : 1 second

Memory Limit : 64 MB
