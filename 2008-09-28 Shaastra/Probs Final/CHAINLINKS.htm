Problems

2. Chain Links

You are given two links in a chain. The chain links are shaped as follows:

  1. Each link is defined by 4 co-planar points in space. Call them points 0, 1, 2 and 3. In order, these points could form the vertex of a convex quadrilateral. Note that the shape of the link is not a convex quadrilateral.
  2. In the link, points 0 and 1 are connected by a line segment. Similarly, points 2 and 3 are connected by a line segment.
  3. Points 1 and 2 are connected by a semicircle with these points as diametric ends. The semicircle is such that it retains the convexity of the link. Points 3 and 0 are connected by a similar semicircle.

We call two links entangled if you cannot separate one link from another (from their given positions) without breaking one of them. Your task is to determine if the given two links are entangled in space.

Input

The first line of input contains the number N of test cases (N <=20). This is followed by N test cases.

Each test case has 8 lines. The first 4 lines contain the x, y and z coordinates of the 4 points defining the first link in order (each line containing coordinates of one point). The next 4 lines contain the coordinates of the 4 points of the second link (-200 <= x <= 200, -200 <= y <= 200, -200 <= z <= 200). All coordinates are integers.

There are no blank lines in the input but there is a newline at the end.

Output

For each test case, you must print Yes in a separate line if the links are entangled in space, and No otherwise.

Sample Input

2 1 2 3 1 3 4 1 3 7 1 2 6 50 57 98 51 45 67 52 45 76 51 57 107 1 2 3 1 3 4 1 3 7 1 2 6 -1 2 5 1 2 5 1 3 6 -1 3 6

Sample Output

No No

Time limit: 1 second

Memory limit: 64 MB
